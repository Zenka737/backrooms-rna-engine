# Backrooms — RNA Engine

Браузерная игра Backrooms на реалистичном движке **RNA (Realistic Navigation & Atmosphere Engine)**.

## Запуск

Открой `index.html` через локальный HTTP-сервер (из-за ES-модулей):

```bash
npx serve .
# или
python3 -m http.server 8080
```

Затем перейди на `http://localhost:8080`.

## Управление

| Клавиша | Действие |
|---------|----------|
| W / ↑ | Вперёд |
| S / ↓ | Назад |
| A | Стрейф влево |
| D | Стрейф вправо |
| ← / → | Поворот |
| Mouse | Поворот (после клика — pointer lock) |
| ESC | Пауза / выход из pointer lock |
| Enter | Начать игру |

## Уровни

| # | Название | Атмосфера |
|---|----------|-----------|
| 0 | The Lobby | Жёлтые обои, мокрый ковёр, монотонный свет |
| 1 | Fluorescent Hum | Бетонные склады, гудящие лампы |
| 2 | Pipe Maintenance | Тёмные тоннели технического обслуживания |

Найди **зелёную точку** на миникарте — это выход на следующий уровень.

## Архитектура RNA Engine

```
src/
  engine/
    RNAEngine.js   — raycasting рендерер, мерцание, туман, виньетка
    Map.js         — процедурная генерация карты (BSP + DDA raycast)
    Player.js      — движение, коллизии, рассудок (sanity)
  levels/
    LevelDefs.js   — определения уровней (цвета, атмосфера, текстуры)
  Game.js          — главный игровой цикл, переходы между уровнями
index.html         — точка входа
```

## Особенности

- **Процедурная генерация** — каждый запуск уникален (BSP-комнаты + коридоры)
- **Реалистичная атмосфера** — мерцание ламп, туман, виньетка, зернистость
- **Санитарный счётчик** — рассудок игрока падает со временем
- **DDA-рейкастинг** — быстрый и точный алгоритм отрисовки стен
- **Миникарта** — помогает не потеряться
